# Öğrenci Not Takip Otomasyonu

Bu proje İstanbul Beykent Üniversitesi Görsel Programlama dersi için geliştirilmiştir.

## Kullanılan Teknolojiler

- C#
- Windows Forms
- SQL Server
- ADO.NET

## Özellikler

- Öğrenci ekleme / silme
- Ders ekleme
- Not giriş sistemi
- Raporlama ekranı

## Durum

Proje geliştirme aşamasındadır.
